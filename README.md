# 🤖 AI Bootcamp Labs

This repository contains a series of structured, hands-on labs exploring how AI can be effectively used in engineering workflows.

The focus is on practical experimentation with prompt engineering, evaluation of AI-generated outputs, and applying AI techniques to real-world scenarios such as software testing.

---

## 🎯 Objectives

The primary goals of this repository are to:

- Demonstrate how prompt design impacts AI output quality  
- Explore iterative approaches to improving AI responses  
- Apply AI concepts to engineering workflows, particularly in testing  
- Establish repeatable patterns for working with large language models  

---

## 🧪 Lab Sessions

Each session focuses on a specific concept and builds progressively:

### ✅ Session 1 — Prompt Basics
- Introduces fundamental prompt design principles  
- Explores how clarity and structure affect AI responses  
- Demonstrates improvement through simple prompt refinement  

---

### 🔁 Session 2 — Prompt Iteration
- Compares multiple prompt variations for the same task  
- Evaluates differences in output quality and consistency  
- Highlights the importance of constraints, roles, and formatting  

---

### 🧪 Session 3 — AI for Testing
- Applies AI to generate software test cases  
- Explores structured output formats (e.g. tables)  
- Demonstrates how AI can support QA workflows and edge case discovery  

---

## 📊 Evaluation Approach

Outputs across sessions are assessed using consistent criteria:

- **Clarity** — Is the response easy to understand?  
- **Relevance** — Does it match the intended context or audience?  
- **Structure** — Is the output organised and usable?  
- **Consistency** — Does the prompt produce reliable results?  

This approach allows for systematic comparison between prompt variations and improvements.

---

## 📂 Repository Structure

This repository is organised by session-based labs, each focused on a specific AI prompt or testing concept.

- `labs/session-01-prompt-basics/`
  - Introductory exercises for writing clear, simple prompts
  - Includes examples, reasoning, and prompt refinement
- `labs/session-02-prompt-iteration/`
  - Iterative prompt design and comparison of prompt variants
  - Highlights how small changes lead to different AI outputs
- `labs/session-03-ai-testing/`
  - Practical QA-focused prompts for generating test cases
  - Contains sample prompt guidance, evaluation, and test case artifacts

Each session folder includes a `README.md` that explains the goal, prompts used, and results from that lab.
